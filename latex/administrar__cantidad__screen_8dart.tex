void \+\_\+confirmar\+Agregar (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Valida la cantidad ingresada y muestra un diálogo de confirmación antes de ejecutar la operación de agregar unidades. 

Regla de negocio\+: no se permite agregar si la cantidad es menor o igual a cero. Al confirmar, delega la operación a \mbox{[}Print\+Provider.\+agregar\+Unidades\mbox{]}, el cual gestiona la impresión de QRs para las nuevas unidades. \Hypertarget{administrar__cantidad__screen_8dart_a5f70c45cd6cdf3d00e67a4a9294141d4}\index{administrar\_cantidad\_screen.dart@{administrar\_cantidad\_screen.dart}!\_confirmarRestar@{\_confirmarRestar}}

Navega a \mbox{[}QRScanner\+Screen\mbox{]} y procesa el resultado del escaneo ejecutando \mbox{[}Restar\+Unidades\+Use\+Case\mbox{]}. 

Mapea cada valor de \mbox{[}Resultado\+Restar\+Unidad\mbox{]} a un \mbox{[}Scan\+Feedback\mbox{]} con el tipo de resultado y mensaje correspondiente.

Reglas de negocio aplicadas\+:
\begin{DoxyItemize}
\item Una unidad ya desactivada no puede restarse de nuevo.
\item Solo se pueden restar unidades que pertenezcan al inventario indicado.
\item El QR debe tener un formato válido reconocido por el sistema.
