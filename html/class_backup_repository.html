Retorna el directorio donde se almacenan los backups dentro del directorio de documentos de la aplicación.

Si el directorio no existe, lo crea de forma recursiva.

◆ crearBackup()

Future< String > BackupRepository::crearBackup ( )
inlineprivate

Genera una copia del archivo de base de datos activo en el directorio de backups.

El nombre del archivo incluye la fecha y hora de creación con formato 'backup_uniformes_YYYY_MM_DD_HH_mm.db' para facilitar su identificación. Retorna la ruta absoluta del archivo de backup generado.
