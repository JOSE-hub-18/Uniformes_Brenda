Descripción detallada

Repositorio encargado de generar consultas estadísticas y métricas utilizadas en reportes.

Centraliza operaciones agregadas relacionadas con ventas, ingresos, prendas y escuelas.

◆ obtenerEscuelasMasVentas()

Future< List< Map< String, dynamic > > > ReporteRepository::obtenerEscuelasMasVentas ( { required int year, int? month, } )
inlineprivate

Obtiene las escuelas con mayores ingresos generados.

La consulta retorna:

  • identificador de escuela,
  • nombre,
  • cantidad de registros vendidos,
  • total monetario generado.

El resultado se limita a las 10 escuelas con mayor ingreso acumulado.

Obtiene las prendas con mayor volumen de ventas.

El resultado se limita a las 10 prendas con mayor cantidad vendida.

◆ obtenerTotalVendido()

Future< double > ReporteRepository::obtenerTotalVendido ( { required int year, int? month, int? idEscuela, } )
inlineprivate

Obtiene el total monetario vendido filtrando por año, mes y escuela de forma opcional.

Las ventas canceladas son excluidas del cálculo.

Cuando se especifica una escuela, el cálculo se realiza utilizando joins sobre detalle_venta, unidades e inventario para identificar únicamente productos asociados.
