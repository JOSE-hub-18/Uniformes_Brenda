Future< int > PrendaRepository::insertar ( Prenda prenda)
inlineprivate

Inserta una nueva prenda en la base de datos.

El identificador primario es eliminado del mapa antes de la inserción para permitir el autoincremento.

La operación utiliza conflicto tipo abort para evitar inserciones duplicadas o inválidas.
