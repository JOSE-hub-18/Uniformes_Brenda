Future< String?> obtenerResumen ( )

Genera un texto resumido con el conteo de alertas activas, apto para mostrar en badges o notificaciones.

Retorna null si no hay alertas activas. Ejemplo de salida: "2 agotados · 5 críticos".

Obtiene del repositorio los items con stock agotado y los mapea a instancias de [AlertaStock] con tipo [TipoAlerta.agotado].

Obtiene del repositorio los items con stock crítico y los mapea a instancias de [AlertaStock] con tipo [TipoAlerta.critico].

◆ verificarItem()

Future< AlertaStock?> verificarItem ( int idInventario)

Verifica si un item específico del inventario tiene una alerta activa.

Consulta el stock actual del item y lo compara contra las reglas de negocio: stock igual a 0 genera alerta de agotado, stock menor o igual a [umbralCritico] genera alerta crítica. Retorna null si el item no tiene alerta activa.
