Descripción detallada

Repositorio encargado de gestionar las operaciones CRUD relacionadas con usuarios del sistema.

Incluye consultas de autenticación lógica, administración de estados y mantenimiento de registros.

◆ insertar()

Future< int > UsuarioRepository::insertar ( Usuario usuario)
inlineprivate

Inserta un nuevo usuario en la base de datos.

El identificador primario es removido del mapa antes de la inserción para permitir autoincremento.

La operación utiliza conflicto tipo abort para evitar registros inválidos o duplicados.
