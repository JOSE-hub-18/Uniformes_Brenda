◆ _conectar()

Future< void > _conectar ( )

Establece la conexión BLE con la impresora.

Busca el dispositivo en la lista de dispositivos vinculados por su dirección MAC, se conecta a él y descubre los servicios y características disponibles. Lanza una excepción si la impresora no está vinculada o si no se encuentra el servicio o la característica requerida.

Busca la impresora en los dispositivos vinculados comparando la dirección MAC de forma insensible a mayúsculas.

Espera breve posterior a la conexión para permitir que el stack BLE estabilice el enlace antes de descubrir servicios.

Itera sobre los servicios descubiertos buscando el UUID de servicio y dentro de él la característica de escritura requerida.

◆ imprimirQR()

override Future< void > imprimirQR ( String id)

Genera y envía un comando TSPL a la impresora para imprimir una etiqueta con código QR e identificador en texto.

El comando se fragmenta en paquetes de 20 bytes con un retardo de 80ms entre cada uno para respetar los límites de transferencia BLE. Lanza una excepción si no hay una conexión activa al momento de imprimir.

Comando TSPL que define el tamaño de la etiqueta (40x30mm), el espaciado entre etiquetas, el código QR y el texto identificador.

Envía los bytes del comando en fragmentos de 20 bytes usando escritura sin respuesta (writeWithoutResponse), con retardo entre fragmentos para evitar desbordamiento en el buffer BLE de la impresora.
